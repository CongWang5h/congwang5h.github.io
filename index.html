Graph

A simple graphical diagram can be useful in describing a (causal) relationship between variables.

Definition

A graph is a collection of vertices or nodes and edges. The nodes are connected (or not) by the edges.

graph LR;
	X --> Y;
	Y --> Z;
  • Two nodes are adjacent if there is an edge between them.
  • The graph is a complete graph if there is an edge between every pair of nodes in the graph
graph LR;
	X --- Y;
	Y --- Z;
	Z ---X
  • A path between two nodes XXX and YYY is a sequence of nodes beginning with XXX and ending with YYY. (each node is connected by an edge)
graph LR;
X --- Z;
Z --- W;
W --- Y
  • Edge in a graph can be directed and undirected.
  • A graph in which all of the edges are directed is a directed graph.
  • Parent. the node that a directed edge starts from is called the parent node of the node that the edge goes into.
  • Child. the node that the edge goes into is the child of the node it comes from.
 graph LR;
 A[Parent]---> B[Child]
  • if two nodes are connected by derected path, the first node is the ancestor of every node on the path and every node is a descendent of the first node.
 graph LR;
 x_1---> x_2
 x_2---> x_3
 x_3---> x_4

  • When a directed graph contains a circle, it is called cyclic.
  • A directed graph with no circles is an acyclic directed graph (DAG).

Structural causal Models

Definition of causation

A variable XXX is a direct cause of a variable YYY if XXX appears in the function that assigns YYY's value.

Y=f(X)Y = f(X) Y=f(X)

XXX is the cause, YYY is the outcome, this function can not inverse, and this process has a chronological order.

Sometimes XXX is not the only cause.

Y=f(X,Z,… ) Y=f(X,Z,\dots) Y=f(X,Z,…)

XXX might be an indirect cause

Association between Causation and Graphs

  • If YYY is the child of XXX. then XXX is the direct cause of YYY.
  • If YYY is the descendent of XXX, then XXX is a potential cause of YYY.

Y is caused by X

graph LR;
X --f--> Y

(YYY is the child of XXX)

Y is caused by X and Z

graph LR;
Z --f-->Y
X --f--> Y

Y is caused by Z, Z is caused by X

YYY is caused by ZZZ, ZZZ is caused by XXX

graph LR;
Z --g-->Y
X --f-->Z

To formally set up the assumptions that the causal relationship behind the data. we introduce Structural Causal Models.

graph LR;
Z --g-->Y
X --f-->Z
W --f-->Z
  • A SCM consists of two sets of variables U={X,W}U=\{X,W\}U={X,W} and V={Z,Y}V=\{Z,Y\}V={Z,Y}, and a sot of functions fff.
  • The variables in UUU are called exogenous variables. (extend to the model)
  • The variables in VVV are called endogenous variables. (internal)
  • Exogenous variables cannot be descendants of any other variables, have no ancestors must be root nodes in graphs.
  • Every endogenous variables is a descendant of at least one exgenous variables.
